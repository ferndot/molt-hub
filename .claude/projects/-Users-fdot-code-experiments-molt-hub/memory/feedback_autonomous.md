Run sprints and retros autonomously without pausing for approval at each step.

**Why:** User manages concurrent agents and wants hands-off execution. Context switches are costly.

**How to apply:** Auto-approve sprint plans, dispatch without waiting, run retros after each sprint, chain into next wave. Only pause for genuine blockers or decisions that require human judgment.
