### Key Decisions
- **3-4 parallel worktree agents per sprint**: Proven reliable across 5 waves. Clean merges except App.tsx (resolved by combining imports).
- **Platform-native UI over custom design**: System fonts, prefers-color-scheme, no manual theme toggle. Sidebar stays dark in both modes (developer tool aesthetic).
- **CSS modules everywhere**: No Tailwind, no CSS-in-JS. CSS custom properties for theme variables.
- **Keyboard nav via CustomEvents**: KeyboardManager dispatches molt:nav-down/up/etc rather than coupling to view stores. Views subscribe independently.
- **css.d.ts required**: Agents creating CSS modules must include type declarations for TypeScript.
- **merge order matters**: Layout shell (T28) must merge before views (T23, T24) to avoid conflicts.
