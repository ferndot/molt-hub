Open Molt Hub — use this if the app does not open automatically (some browsers block custom URL schemes unless you click a link).

If nothing happens, open the app manually — it must be running. On macOS, tauri dev alone often does not register the molthub:// handler; build the app once with cargo tauri build --debug and open the .app from target/debug/bundle/macos/ (or install it) so the system knows to hand off molthub://.
